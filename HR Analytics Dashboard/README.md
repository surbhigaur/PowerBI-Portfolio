# HR Analytics Dashboard | Power BI

## 📌 Project Overview

This HR Analytics Dashboard was developed using Power BI to analyze employee data and understand workforce trends. The dashboard provides insights into employee demographics, attrition patterns, job roles, and key HR metrics.

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

## 📊 Dashboard Features

- Employee Count Analysis
- Attrition Analysis
- Department-wise Employee Distribution
- Gender & Age Analysis
- Job Role Analysis
- Salary & Experience Insights
- Interactive HR Reports
