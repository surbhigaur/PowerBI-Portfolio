# Top Line Report | Power BI

## 📌 Project Overview

This Top Line Report dashboard was created using Power BI to analyze overall business performance and provide key insights through interactive visualizations.

The dashboard helps track important business KPIs, sales trends, and performance metrics for better decision-making.

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

## 📊 Dashboard Features

- Key Performance Indicators (KPIs)
- Business Performance Analysis
- Sales Trend Analysis
- Interactive Charts and Slicers
- Performance Monitoring Dashboard
