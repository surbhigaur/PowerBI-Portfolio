# Ecommerce Sales Dashboard | Power BI

## 📌 Project Overview

This Ecommerce Sales Dashboard was created using Power BI to analyze online sales performance and understand key business metrics. The dashboard provides insights into revenue trends, customer behavior, product performance, and sales patterns.

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

## 📊 Dashboard Features

- Total Sales & Revenue Analysis
- Profit Analysis
- Category-wise Sales Performance
- Product Performance Analysis
- Customer Insights
- Regional Sales Analysis
- Monthly Sales Trends
- Interactive Filters and Slicers
