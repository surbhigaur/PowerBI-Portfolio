# Online Sales Dashboard 2022-2023 | Power BI

## 📌 Project Overview

This Online Sales Dashboard was developed using Power BI to analyze online sales performance for 2022–2023. The dashboard provides interactive insights into revenue, order quantity, customers, products, brands, payment methods, and regional sales performance, helping businesses monitor KPIs and make data-driven decisions.

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

## 📊 Dashboard Features

- Revenue & Sales Performance Overview
- Total Orders, Products & Customers Analysis
- Brand-wise Revenue, Sales & Order Quantity
- State-wise & City-wise Sales Performance
- Customer-wise Revenue Analysis
- Payment Method Analysis
- Interactive Filters (Year, Month, Date, Customer, State & City)

## 📈 Key KPIs

- Total Revenue
- Total Sales Amount
- Total Orders
- Total Order Quantity
- Total Customers
- Repeated Customers
- Product Count
