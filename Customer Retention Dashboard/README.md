# Customer Retention Dashboard | Power BI

## 📌 Project Overview

This Customer Retention Dashboard was developed using Power BI to analyze customer behavior, identify churn patterns, and understand the key factors affecting customer retention. The dashboard enables businesses to monitor customer demographics, service usage, revenue, and retention metrics through interactive visualizations.

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

## 📊 Dashboard Features

- Customer Profile Overview
- Total Customers, Revenue & Average Charges
- Customer Churn Rate Analysis
- Payment Method Distribution
- Contract Type Analysis
- Internet Service Analysis
- Customer Demographics
- Customer Retention Insights
- Interactive Filters and Slicers
