# Call Center Dashboard | Power BI

## 📌 Project Overview

This Call Center Dashboard was developed using Power BI to monitor call center performance and provide insights into customer service operations. The dashboard helps analyze call volumes, agent performance, customer satisfaction, and key service metrics to support operational decision-making.

## 🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

## 📊 Dashboard Features

- Total Calls Analysis
- Customer Satisfaction (CSAT) Analysis
- Call Resolution Metrics
- Agent Performance Tracking
- Call Volume Trends
- Call Duration Analysis
- Interactive Filters and Slicers
